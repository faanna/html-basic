div태그는 html문서에서 특정영역이나 구획을 정의할 때 사용합니다. 이외에도 아주 다방면으로 활용이 가능합니다
span태그는 inline속성을 가진 div태그라고 할수 있습니다. 아주 다방면으로 활용이 가능합니다

p태그는 하나의 문단을 만들때 사용하는 태그입니다

    ul태그는 순서가 없는 리스트 태그입니다
    ol태그는 순서가 있는 리스트 태그입니다
  • li태그는 ul,ol태그 안쪽에 사용되는 목록태그입니다
    • dl태그는 용어와 그에대한 설명을 리스트 형식으로 정의하는 태그입니다
    dt태그는 dl태그안에서 정의하는 용어의 제목을 넣을때 사용하는 태그입니다
    dd태그는 dl태그 안에서 용어을 설명하는데 사용하는태그

    h1태그는 해당 영역의 제목을 나타내며 해당 영역에서 한번만 사용합니다

    h2태그는 h1과 같이 제목태그이며, h1을 사용할수 없을때 사용하는 태그입니다

    h3태그도 제목태그로 하위 컨텐츠의 제목이나 좀더 작은 개념의 제목을 나타낼때 사용하는 태그입니다 h6까지 존재합니다

    이태그는 주소를 나타낼 때 사용합니다
    a태그는 링크를 시킬수 있습니다, 기본적으로 텍스트스타일이 입혀져 있습니다
    strong태그는 강조하는 용도(중대,긴급)로 사욛되며 굵은 텍스트로 표시됩니다
    b태그는 단순하게 굵게 표현되는 텍스트입니다
    em태그는 문장내에 특정 문맥을 강조할 때 기울여서 표현됩니다
    i태그는 텍스트에서 주위와 구분해야하는 특정 부분을 기울여서 나타냅니다(예: 기술용어,외국어구절등)
